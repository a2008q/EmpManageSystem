# EmpManageSystem 员工管理系统

## 前端
### 前端使用vue_cli脚手架完成，需安装好相关依赖
```
cd ssm_crud_web
npm install
```

### 开发模式运行前端
```
npm run serve
前端地址 http://127.0.0.1:8080
```

### 编译发布
```
npm run build
```
